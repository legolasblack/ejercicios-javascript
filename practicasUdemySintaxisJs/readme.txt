En esta Archivo vamos a hacer un indice de practicas y recopilado de tecnicas necesarias para  consultas 
posteriores propias, 
Instrucciones:

1.- El Archivo Index funciona para poder abrir dedntro del navegador y se tratara de que todo 
corra por medio de la consola

2.- Para correr cualquier archivo solo es necesario invocar el archivo js dentro del index y ejecutar con el navegador.

3.- Si la practica esta en TypeScript, es necesario primero copilar el archivo ts a js y despues ejecutarlo en el index.
